# Atividade Prática - Fundamentos da Programação Web 🚀

Repositório destinado ao desenvolvimento do projeto prático final da disciplina de Fundamentos da Programação Web, integrante da grade curricular do curso de Análise e Desenvolvimento de Sistemas (UNINTER).

## 📋 Objetivo do Projeto
Desenvolver um portfólio pessoal e profissional online (currículo web de página única com navegação por âncoras) utilizando exclusivamente tecnologias nativas do ecossistema web, sem a utilização de frameworks CSS ou bibliotecas JavaScript.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica de todo o conteúdo da página única.
* **CSS3:** Estilização e diagramação visual com foco em responsividade e harmonia de cores.
* **JavaScript (Vanilla):** Lógica nativa de validação do formulário de contato, tratamento de eventos e simulação interativa de envio.

## 📂 Estrutura de Páginas/Seções Desenvolvidas
Em conformidade com os requisitos obrigatórios descritos nas diretrizes da instituição, o portfólio contempla as seguintes seçõe:
1. **Sobre Mim:** Informações e resumo de apresentação profissional.
2. **Formação:** Percurso acadêmico e cursos complementares de qualificação tecnológica.
3. **Portfólio:** Galeria de projetos desenvolvidos e tecnologias em estudo.
4. **Contato:** Formulário com validação em tempo de execução via JavaScript.
