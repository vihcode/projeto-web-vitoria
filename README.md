# Atividade Prática - Fundamentos da Programação Web 🚀

Repositório destinado ao desenvolvimento do projeto prático final da disciplina de Fundamentos da Programação Web, integrante da grade curricular do curso de Análise e Desenvolvimento de Sistemas (UNINTER)[cite: 2].

## 📋 Objetivo do Projeto
Desenvolver um portfólio pessoal e profissional online (currículo web de página única com navegação por âncoras) utilizando exclusivamente tecnologias nativas do ecossistema web, sem a utilização de frameworks CSS ou bibliotecas JavaScript[cite: 2].

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica de todo o conteúdo da página única.
* **CSS3:** Estilização e diagramação visual com foco em responsividade e harmonia de cores[cite: 1, 2].
* **JavaScript (Vanilla):** Lógica nativa de validação do formulário de contato, tratamento de eventos e simulação interativa de envio[cite: 2].

## 📂 Estrutura de Páginas/Seções Desenvolvidas
Em conformidade com os requisitos obrigatórios descritos nas diretrizes da instituição, o portfólio contempla as seguintes seções[cite: 2]:
1. **Sobre Mim:** Informações e resumo de apresentação profissional[cite: 2].
2. **Formação:** Percurso acadêmico e cursos complementares de qualificação tecnológica[cite: 2].
3. **Portfólio:** Galeria de projetos desenvolvidos e tecnologias em estudo[cite: 2].
4. **Contato:** Formulário com validação em tempo de execução via JavaScript[cite: 2].
